# CatPhotoApp 🐱

A simple cat photo app built with pure HTML as part of the [freeCodeCamp](https://www.freecodecamp.org) Web Development curriculum.

## About

CatPhotoApp is a beginner HTML project that showcases cat photos, fun cat facts, and an interactive cat submission form.

## Features

- 🖼️ Cat photo gallery with links to cute cat images
- 📋 Lists of things cats love and hate
- 📝 A form to submit your own cat photo — including indoor/outdoor selection and personality checkboxes

## Built With

- HTML5 (semantic elements: section, figure, fieldset, footer and more)

## What I Learned

- Structuring a webpage with semantic HTML
- Working with images, links, and figures
- Building forms with radio buttons, checkboxes, and text inputs
